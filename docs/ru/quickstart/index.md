# :material-rocket-launch: Быстрый старт

!!! tip "Эта страница — для тех, кому нужно быстро"

    Минимум объяснений, максимум действий. Если хотите понять **зачем** —
    читайте полные разделы [Маршрутизация](../routing/index.md) и
    [DNS](../dns/index.md).

---

## 1. Маршрутизация

**Настройка → Настройки маршрутизации** → добавьте правила или импортируйте
готовый JSON:

??? note "JSON правил маршрутизации (нажмите, чтобы развернуть и скопировать)"

    ```json
    --8<-- "configs/v2rayn/xray/routing-rules.json"
    ```

---

## 2. DNS для Xray

**Настройка → Настройки DNS V2ray** → вставьте:

??? note "JSON DNS Xray (нажмите, чтобы развернуть и скопировать)"

    ```json
    --8<-- "configs/v2rayn/xray/dns.json"
    ```

---

## 3. DNS для sing-box (если используете TUN)

**Настройка → Настройки DNS sing-box** → вставьте:

??? note "JSON DNS sing-box (нажмите, чтобы развернуть и скопировать)"

    ```json
    --8<-- "configs/v2rayn/sing-box/dns.json"
    ```

---

## 4. Проверка

<div class="steps" markdown>

1. Перезапустите соединение: кнопка Reload в меню
2. Откройте [dnsleaktest.com](https://dnsleaktest.com) → Extended test
3. В результатах должны быть **только** Quad9 и/или Mullvad
4. Откройте [2ip.ru](https://2ip.ru) → должен показать **ваш российский IP**
5. Откройте [youtube.com](https://youtube.com) → должен открыться через прокси

</div>

Если всё работает — :tada: поздравляю! Если нет — смотрите
[подробную проверку](../verification/index.md) и [FAQ](../faq/index.md).
